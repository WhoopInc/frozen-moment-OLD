FrozenMoment

FrozenMoment

Parse, validate, manipulate, and display dates in JavaScript.
Immutable fork of Moment.js, brought to you by Whoop.

Download

frozen-moment.js

Install

bower install --save https://github.com/WhoopInc/frozen-moment.git # bower
npm install git+https://github.com/WhoopInc/frozen-moment.git --save   # npm

Format Dates

frozenMoment().format('MMMM Do YYYY, h:mm:ss a');
frozenMoment().format('dddd');
frozenMoment().format("MMM Do YY");
frozenMoment().format('YYYY [escaped] YYYY');
frozenMoment().format();

Relative Time

frozenMoment("20111031", "YYYYMMDD").fromNow();
frozenMoment("20120620", "YYYYMMDD").fromNow();
frozenMoment.build().startOf('day').freeze().fromNow();
frozenMoment.build().endOf('day').freeze().fromNow();
frozenMoment.build().startOf('hour').freeze().fromNow();

Calendar Time

frozenMoment.build().subtract(10, 'days').freeze().calendar();
frozenMoment.build().subtract(6, 'days').freeze().calendar();
frozenMoment.build().subtract(3, 'days').freeze().calendar();
frozenMoment.build().subtract(1, 'days').freeze().calendar();
frozenMoment().calendar();
frozenMoment.build().add(1, 'days').freeze().calendar();
frozenMoment.build().add(3, 'days').freeze().calendar();
frozenMoment.build().add(10, 'days').freeze().calendar();
